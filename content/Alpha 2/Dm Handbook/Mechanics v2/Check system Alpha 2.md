# What is a check
To start lets explain what a check is.

A check is just a phrase for when players do something that would need to be checked, Such as trying to convince an NPC of something or getting information could use the Communication Core Attribute or The Persuasion Skill Attribute. 

Where you would roll a d10 and add the Attribute ontop of the roll. 


# What is an attribute?

While this is explained in  [[Character Sheets Alpha]] I will re explain them here,

#### Core Attributes
Strength - Can be used for feats of strength, Ex: Breaking down a door.

Dexterity - Can be used for feats of dexterity, Sprinting down multiple hallways, Traveling between towns. Ect.

Energy - Determines the amount of moves an npc/Anomaly/Player can do in a turn during combat, Plans are also to have it affect things outside of combat but that has not been worked on
