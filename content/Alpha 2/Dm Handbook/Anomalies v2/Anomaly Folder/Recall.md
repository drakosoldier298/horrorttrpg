A small Old fashioned HourGlass, Adorned with glittering gold, the sand is a bright blue and pulsating as if emanating an energy,

Taped to the side is a small label that says "Recall" 

### Usage
A small slip of paper is to be placed in the empty side of the hourglass with the users name on one side and the time on the other, This will save their **exact** location and state.

When flipped it will start a timer that lasts <font color="#c00000">Four hours</font> where when the sand fully covers the piece of paper it will recall the user to the exact location and state they were in when writing the paper. 

User can manually activate the affect by shaking the Hour Glass with the intent to recall. 

Upon manually activating the effect the hourglass will enter cooldown mode where the glass will turn black and it will stop glowing, This lasts <font color="#c00000">Two hours</font>. 

It can hold up to Two pieces of paper, When the effect is activated the user MUST be thinking of which state they want to return to. If the user is not it will split them into two people at both states. It is up to the GM what happens in this instance, One example is that the player now can control 2 copies of themself, another is that they flip a coin and control whichever it lands on and the GM controls the other.
