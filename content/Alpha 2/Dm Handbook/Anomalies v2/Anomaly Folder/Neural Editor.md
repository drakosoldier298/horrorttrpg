This Anomaly is derived From the Memory gun in Gravity Falls


#### Description
A cobbled together gun, its a mess of copper pipes, wires and a Lightbulb, Chaotic yet well crafted.

The main body is a copper Tube with a hinge for a "cartridge" to be placed inside it, with the end of the gun being a pointed lightbulb, on the left side is a small keyboard and screen that can be folded out for use.

###### Cartridge 

A glass vial Reinforced with copper.


#### Usage
To use the device simply insert a cartridge and fold out the keyboard and screen, type in what you want the victim to forget and pull the trigger. 

The cartridge will have a glowing electricity in it if successful, This is the memory, if destroyed it is gone forever.

**Facility note** - We have noticed that with some use cases, Such as making A victim forget themself, The affect is temporary, Removing large amounts of memory seems to also be temporary. 

More testing needs to be done.
