## Description
A relatively small device the size of a printer with 2 compartments, With rough labels marking the left as input and the right as output

A small Rusted dial is in between the 2 compartments with printed labels reading "<font color="#c00000">Rough</font>", "<font color="#00b050">1:1</font>", "<font color="#0070c0">Fine</font>"  
## Use case
When an item is held near the input it will shrink the item so it fits into the compartment, It seems to know the intent of the user and does not shrink items that the user doesn't intend to be put in the compartment.

Depending on the setting of the dial The anomaly will do different things, 
##### Single object
If a single object is placed inside and it is set to <font color="#00b050">1:1</font> it will split into the direct parts used to create the object <font color="#e36c09">Example</font>: a Computer would be turned into A circuit board, Graphics card, Iron sheets (case) Ect.
If set to <font color="#c00000">Rough</font> it will be the rough parts used to create the object, <font color="#e36c09"><font color="#e36c09">Example</font></font>: A computer placed inside will be turned into copper, Gold Iron and rubber
If set to <font color="#0070c0">Fine</font> It will be turned into a better version of what was placed inside, <font color="#e36c09">Example</font>: a computer Would turn into a super computer. 

##### Multiple Objects

If multiple objects are placed inside On <font color="#00b050">1:1</font> It will directly combine them, If no glue or adhesive was put inside it will create duct tape that would fall apart the second it is attempted to be removed

If set to <font color="#c00000">Rough</font> The objects will all be turned into their rough parts.
 
If set to <font color="#0070c0">Fine</font> It will combine the objects in a way that does not need glue or adhesive, Most of the time resulting in a better product Example,  spoon and a fork placed inside creates a spork.
