# Important
#### This is an example list, To be used as a way to show how to make your own creatures, or to be used if you don't want to make your own.

**To players**
This list isn't the required monsters, and the values can be changed at any time. Do not take the values as fact. 

**To Game Masters**
Please feel free to edit values or lore for your own universe. I actually encourage it so that your players do not fully know what to expect when they read, To many players read The DM book and go "well now I know what to do" Causing An issue with metagaming.
